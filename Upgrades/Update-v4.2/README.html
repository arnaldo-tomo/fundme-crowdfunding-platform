Installation

IMPORTANT: You must update from version 4.1, if you upgrade from another version will throw an error.
  • Upload the folder v4.2 to the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Enter this URL https://yoursite.com/v4.2/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v4.2/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
  • The file affected with the new text strings is: resources / lang / en / misc.php
Note: If you want to add a new language, read the documentation.

Installation

  • Upload the folder v3.6 to the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Enter this URL https://yoursite.com/v3.6/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v3.6/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
  • Finally add these new text strings in your file resources / lang / en / misc.php
